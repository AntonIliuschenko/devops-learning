# Day 2 — Filesystem

## Linux filesystem structure

/
├── home
├── etc
├── var
├── usr
├── tmp

## Key directories

- /home — user files
- /etc — configuration
- /var — logs and variable data
- /tmp — temporary files

## File operations

```bash
touch file.txt
mkdir dir
cp file1 file2
mv file1 file2
rm file.txt
rm -r dir

# Important

Be careful with rm -rf (can delete everything).
