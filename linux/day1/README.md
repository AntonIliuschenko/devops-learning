# Day 1 — Shell basics

## What is shell
Shell is a command-line interface to interact with the Linux system.

## Basic commands

```bash
pwd     # current directory
ls      # list files. Arguments -l list, -la allfiles(hidden)
cd      # change directory. Arguments .. level up, ~/ home folder, / root folder
clear   # clear screen
history # previous commands

```

## File paths
Absolute: /home/user/file
Relative: ./file
Important concept

## 🧠 Important concept
> Everything in Linux is a file.
